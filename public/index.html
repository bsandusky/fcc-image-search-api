FCC Image Search API

This page redirects to an image search API which runs a search on the public gallery of Imgur. This project is the FCC Image Abstraction Layer project; more info can be found here.

To use the API directly, navigate to /api/search/ and add a query and offset in the following format ?q=query&offset=10. The offset will default to 1 result, but a query is required to the API to complete the call.

To view the code, visit the github page here.
